PROJECTS

The Effect of Code Dependencies on Software Project Quality

At NC State, I researched how dependencies in software development projects affect project quality. By analyzing open source software project data from IBM, I identified technical and social dependencies and measured their effect on issue resolution in the project. I found that social dependencies related to software teams and their interactions with the project were much better predictors of the issue resolution type than technical dependencies. I presented my poster at the Undergraduate Research Symposium at the end of the summer.

Twitter Mining the 2016 Presidential Election

Using Twitter's API in Python, my group and I collected over 55,000 tweets from February 23rd, 2016 related to keywords about the top five presidential candidates. Using natural language processing, we cleaned dataset to identify key words, and constructed wordclouds in R to visualize the top results for each candidate.

Climate Change Survey Response Mapping

Using data from a partner who worked at Pew Research Center, we mapped online (non-random, non-probabalistic) survey responses to questions about the severity of the problem of climate change. We used the IP addresses to obtain latitude and longitude points in Python, and used Google Maps' API in R to visualize the responses by region. The goal was to measure attitudes about climate change based on geographical region.
